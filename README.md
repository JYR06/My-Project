# My-Project1_System Archive Terminal v4.2

💡 Project Philosophy
단순한 기록을 넘어 '나라는 시스템의 버전 관리'를 목표로 합니다.
감정적인 서술에 치우치기보다 객관적인 지표(KPI)와 시각적 증거(Visual Record)를 통해 스스로를 객관화하고, 다음 스프린트의 전략을 수립합니다.

✨ Key Updates (v4.2)
1. Portable Single-File Architecture
- 별도의 개발 환경 구축이나 npm install 없이, index.html 파일 하나만으로 즉시 실행 가능합니다.
- React, Tailwind CSS, Lucide Icons를 CDN 기반으로 통합하여 어디서든 브라우저만 있으면 접속할 수 있습니다.
2. Monthly Visual Archive (New)
- 텍스트 위주의 회고에 시각적 깊이를 더했습니다.
- 매달 가장 성취감이 컸던 순간이나 프로젝트 결과물을 최대 4장의 사진으로 박제하여 '성장의 증거'를 남깁니다.
3. Dynamic Category Management
- 고정된 항목이 아닌, 사용자의 현재 관심사에 따라 점검 카테고리를 자유롭게 추가하거나 삭제할 수 있습니다.
4. Custom Visual Identity
- 직접 스케치한 '시그니처 페이스' 로고를 SVG로 디지털화하여 적용했습니다.

🛠 Features
- KPI Dashboard: 핵심 성과 지표를 실시간 슬라이더로 조절 및 시각화
- KPT Framework: Keep(강점), Problem(병목), Try(액션 플랜) 기반의 전략 도출
- Monthly Selector: 과거 기록을 언제든 불러오고 새 달을 추가하는 아카이브 기능
- Data Persistence: Browser LocalStorage를 통한 데이터 영구 보존

🚀 How to Use
- 본 레포지토리를 클론하거나 index.html 파일을 다운로드합니다.
- index.html을 크롬이나 웨일 등 브라우저로 엽니다.
- 우측 상단의 편집(연필) 아이콘을 눌러 데이터를 입력하고 하단의 업데이트 버튼을 눌러 저장합니다.
